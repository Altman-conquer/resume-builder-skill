# License Checklist

Use this checklist before fetching, copying, modifying, or redistributing a third-party resume template.

## Required Checks

1. Find the upstream license file or project license metadata.
2. Confirm whether redistribution, modification, and commercial use are allowed.
3. Preserve the upstream license file when copying template source.
4. Preserve author attribution in README, comments, or generated output when required.
5. Check whether fonts, screenshots, sample PDFs, and icons have separate licenses.
6. Record uncertainty instead of assuming permission.

## Decision Rules

- Clear permissive license: use the template while preserving attribution and license text.
- Copyleft license: explain obligations before mixing with other files.
- No license file: do not vendor the source; use as external reference only unless the user explicitly accepts the risk.
- Separate font or image license: keep those assets external unless redistribution is clearly allowed.

## User-Facing Summary

When license details matter, report:

- upstream project URL;
- detected license;
- whether the source can be copied;
- required attribution;
- any unresolved uncertainty.
